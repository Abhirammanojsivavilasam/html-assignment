    FIVE HTML TAGS

    label
    label tag goes with every field, for Accessibility
    input
    input tag is the most common form tag. It has many “types” including text, number, email, checkboxes, radio buttons password, date, range, even color!
    textarea
    textarea is for larger blocks of multi-line text
    button
    button tag will activate the form that will send data to the server
    table
    table tag is used for create table
